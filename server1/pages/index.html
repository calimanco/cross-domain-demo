Cross Origin Demo Index

The following demos will use cross-domain technology to request API of server2.

  • SubHostProxy（子域名代理）
  • HTML-P/MockForm（自填充HTML/模拟表单）
  • WindowName
  • WindowHash
  • PostMessage
  • CORS / cross-origin resource sharing（跨源资源分享）
